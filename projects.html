Projects

Mystery Dungeon Clone

My first large scale project was a clone of the popular series 'Pokemon Mystery Dungeon' made with the popular game framework libGDX. It's how I learned how to apply Object Oriented Principles and Design Patterns.

The project is currently on hiatus in order to pick up different languages and get some new perspectives, but I will come back to it.

Personality Tester

This was a small project working with libGDX's scene2d api, where I learned more about extensibility, design structure and reading and storing data.

It asks a series of questions with multiple answers, with each answer having a different weight. At the end, the program gives a very rough outline of your personality.

It's basically finished, but new questions and answers can easily be added through JSON.
